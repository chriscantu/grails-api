REST-API
=============

# Overview
Simple Grails 3.0 proof of concept RESTful api.

## Setup
* install grails 3.0 or above
* `git clone git@github.com:chriscantu/grails-api.git`
* `cd grails-api`

## Running
* `grails run-app`

## Testing
* `grails test-app`

## Building
* `gradle build`

## Deploying
* `gradle war`
